# Conditional Sentences

Conditional sentences describe situations that depend on a condition. They come in several types depending on how real or likely the condition is.

## The Four Conditionals

| Type | If-clause | Main clause | Use |
|---|---|---|---|
| **Zero** | Present simple | Present simple | General truths |
| **First** | Present simple | *will* + base verb | Real future possibility |
| **Second** | Past simple | *would* + base verb | Unreal/hypothetical present |
| **Third** | Past perfect | *would have* + past participle | Unreal past (can't change) |

### Zero Conditional — Facts
✅ If you **heat** water to 100°C, it **boils**.
✅ When it **rains**, the ground **gets** wet.

### First Conditional — Real Future
✅ If it **rains** tomorrow, we **will stay** inside.
✅ She **will pass** the exam if she **studies**.
❌ If it will rain tomorrow... *(no future tense in if-clause)*

### Second Conditional — Unreal Present
✅ If I **had** a million dollars, I **would travel** the world.
✅ If she **were** here, she **would know** what to do.
❌ If I would have a million dollars... *(wrong structure)*

### Third Conditional — Unreal Past
✅ If I **had studied** harder, I **would have passed**.
✅ She **would have come** if you **had invited** her.
❌ If I would have studied... *(colloquial, not formal)*

## Mixed Conditionals

Combine the if-clause from one type with the main clause from another:

✅ If I **had studied** medicine *(past)*, I **would be** a doctor now *(present)*.
✅ If she **were** more organized *(present state)*, she **would have finished** on time *(past)*.

## *Unless*, *Provided That*, *As Long As*

| Word | Meaning | Example |
|---|---|---|
| unless | if ... not | ✅ You'll fail **unless** you study. |
| provided (that) | only if | ✅ **Provided that** it doesn't rain, we'll go. |
| as long as | on condition that | ✅ You can go **as long as** you're back by 10. |

## Common Errors

| Error | Correction |
|---|---|
| If I **will see** her, I'll tell her. | If I **see** her, I'll tell her. |
| If I **would have** known... | If I **had** known... |
| If I **was** rich... | If I **were** rich... *(subjunctive)* |
| I wish I **would have** gone. | I wish I **had** gone. |

✅ If I had known you were coming, I would have baked a cake.
❌ If I would have known you were coming, I would have baked a cake.
✅ If I were taller, I would play basketball.
❌ If I was taller, I would play basketball.
