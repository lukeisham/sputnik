# Irregular Plurals

Most English nouns form plurals by adding *-s* or *-es*. Irregular plurals break this rule — inherited from Old English, Latin, Greek, and other sources.

## Common Irregular Patterns

### Vowel Change (Germanic)
| Singular | Plural |
|---|---|
| man | men |
| woman | women |
| foot | feet |
| tooth | teeth |
| goose | geese |
| mouse | mice |
| louse | lice |

### Latin and Greek Endings
| Ending | Singular | Plural |
|---|---|---|
| -us → -i | alumnus, cactus, focus | alumni, cacti, foci |
| -um → -a | datum, medium, bacterium | data, media, bacteria |
| -a → -ae | formula, antenna, larva | formulae/ulas, antennae/as, larvae |
| -is → -es | crisis, thesis, axis | crises, theses, axes |
| -on → -a | phenomenon, criterion | phenomena, criteria |
| -ex/-ix → -ices | index, appendix, matrix | indices/ixes, appendices/ixes, matrices |

### No Change
| Singular and Plural Same |
|---|
| sheep, deer, fish, species, series, aircraft, moose, bison, salmon, trout |

### *-en* Plurals (Old English)
| Singular | Plural |
|---|---|
| child | children |
| ox | oxen |
| brother | brethren *(archaic; "brothers" is standard)* |

## Edge Cases and Common Mistakes

### Compound Nouns
✅ mothers-in-law, passers-by, attorneys general
❌ mother-in-laws, passer-bys *(pluralize the main noun)*

### Words Ending in *-f* or *-fe*
| Singular | Plural |
|---|---|
| knife | knives |
| wife | wives |
| wolf | wolves |
| leaf | leaves |
| **but:** roof | roofs |
| **but:** chief | chiefs |
| **but:** belief | beliefs |

### Mass Nouns (No Plural Form)
✅ information, furniture, advice, equipment, luggage, knowledge
❌ informations, furnitures, advices

### Plural-Only Nouns
✅ scissors, pants, glasses, pliers, binoculars *(always plural)*
*"A pair of scissors"* makes them countable.

### Data — Singular or Plural?
- **Traditionally:** *data* is plural (*datum* is singular).
  ✅ The data **are** conclusive. *(formal/academic)*
- **Modern:** *data* as a mass noun is widely accepted.
  ✅ The data **is** conclusive. *(common usage)*

### Apostrophe Errors
✅ The 1990s, CDs, FAQs, IOUs
❌ The 1990's, CD's, FAQ's *(never use apostrophe for regular plurals)*
